# Prueba técnica Ditech

Aplicación full-stack (Angular + NestJS) para cargar candidatos ví­a formulario, procesar un Excel de una fila y mostrar el listado combinado.

Para levantar Front puedes utilizar el comando "npm run start:frontend"
Para levantar Back puedes utilizar el comando "npm run start:backend"

Para levantar todo el proyecto puedes utilizar el comando "npm run start:all"

Por defecto los puertos utilizados son los habituales --> 4200 para Front y 3000 para Back
